Buscar no mínimo um artigo científico que trate do uso de pelo menos uma estratégia de redes complexas (hubs, comunidades, motifs, etc.) para analisar dados relacionados à expressão gênica. A análise do artigo deve estar alinhada com o tema com que a equipe trabalha. O subgrupo deve escrever um resumo de no mínimo meia página e no máximo uma página, com no mínimo uma ilustração apresentando a abordagem adotada. Esta ilustração, ou outra adicional, deve apresentar o modelo de grafo usado no trabalho. A explicação deve detalhar que estratégia de grafo foi usada sobre o modelo de grafo escolhido para dar suporte a descobertas.

Como cada equipe é formada por dois subgrupos, haverá no mínimo a análise de dois artigos por equipe. Esta análise será incorporada na fundamentação teórica do projeto da equipe. Mesmo assim, neste estágio, cada subgrupo deve apresentar sua contribuição separadamente.

Foi estabelecido o número mínimo de artigos, figuras e tamanho. O subgrupo pode decidir ampliar este número se achar pertinente, mas deve sempre se ater a apresentar os resultados de forma resumida.

O resultado deve ser apresentado na forma de um markdown no GitHub da equipe (neste caso, deve haver uma página por subgrupo) ou em um PDF, que deve ter no mínimo:

- sigla da equipe
- nome dos ~3 membros do subgrupo
- resumo descritivo e ilustrações
- referência(s)
